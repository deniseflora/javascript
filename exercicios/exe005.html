Operadores em Javascript

  • Operador + Adição Ex: 3+ 2 retorna 5
  • Operador - Subtração Ex: 3 – 2 retorna 1
  • Operador * Multiplicação Ex: 3 * 2 retorna 6
  • Operador / Divisão Ex: 3 / 2 retorna 1.5
  • Operador % Módulo (resto da divisão) Ex: 12% 5 retorna 2
  • Operador **
    • Exponenciação Ex: 2**3= 8

Ordem de Precedência dos Operadores Aritméticos

  • ()
  • **
  • *, /, % (preferência da esquerda para direita
  • +, -

Auto-atribuições

  • var n= 3
  • n= n+ 4
  • n= n- 5
  • n= n* 4
  • n= n/ 2
  • n= n** 2
  • n= n% 5

Simplificando

  • n += 4
  • n -= 5
  • n *= 4
  • n /= 2
  • n **= 2
  • n %= 5

Operadores de Incremento

  • var = 5
  • x = x+1
  • x = x-1

Simplificando

  • x ++
  • x --
