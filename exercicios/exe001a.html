Variáveis e Tipos Primitivos

  1. Definição

    • Uma variável é uma posição na memória que permite armazenar dados de forma temporária.

  2. Regras de nomeação

    • Começar comletra, undescore(_) cifrão($).
    • Não pode começar por números
    • Não são permitidos espaços ou caracteres especiais
    • Notação CamelCase para nomes compostos (por ex.:, nomeCliente)
    • Nomes de variáveis são case-sensitive
    • Não declarar com palavras reservadas

  3. Tipos
    • String (Texto)
    • Number (Subtipos: Int(inteiros),Float(fracionados)
    • Boolean (True ou False)
    • Null (Nulo)
    • Undefined (Indefinido)

  4. Declaração de uma variável
    • palavra var
    • palavra let
