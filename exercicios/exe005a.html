Operadores Relacionais

  • Operador maior (>) Ex: 5>7 False
  • Operador menor (<) Ex: 5<7 True
  • Operador maior ou igual (>=) Ex: 8>=8 True
  • Operador menor ou igual (<=) Ex 5<=7 False
  • Operador Igualdade == Ex 5==5 True
  • Operador Desigualdade != Ex 6!=7 True

Operadores de Identidade

  • Operador Igual(mesma grandeza, não testa o tipo)== Ex 5=='5' TrueEx 5==5 True
  • Operador Idêntico (mesmo valor e tipo)=== Ex 5==='5' False Ex 5===5 True

Operadores Lógicos

  • Operadores de Negação !
  • Operadores de Conjunção &&
  • Operadores de Disjunções ||

Operadores de Negação

  • ! True --> False
  • ! False --> True

Operadores de Conjunção

True só com 2 verdadeiras

  • True&&True--> true
  • True&&False-->False
  • False&&True-->False
  • False&&False-->False

Operadores de Disjunção

True basta 1 verdadeira

  • True||True--> true
  • True||False-->true
  • False||True-->true
  • False||False-->False
